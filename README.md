# Ekinexpo Website

Corporate website project developed for Ekinexpo.

## Technologies

* Next.js
* TypeScript
* Tailwind CSS
* React

## Features

* Responsive design
* Service pages
* Project showcase
* Contact form
* SEO-friendly structure

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Status

Project is currently under active development.
